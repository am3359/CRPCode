所有任务一次创建完成，每个任务是一个状态机，有定时复位机制

任务：相互之间是否有冲突
Main：硬件初始化，创建主任务start_task，启动调度器
主任务：(start_task) 创建软件定时器，消息队列，信号量，事件标志组，任务通知等，创建完成删除自己。
HMI界面UART通讯：(hmi_task)负责界面显示。发送命令给HMI界面，接收HMI界面消息，接收其它任务的消息

串口通讯：(com_task)接收控制命令，返回信息。接收串口消息，发送消息给串口

阀控制、步进电机控制、泵控制：(valve_task，stepmoto_task，pump_task)接收阀控制信息（开关），接收步进电机控制信息（速度，方向，位置），接收泵控制信息（速度，方向，时间）；返回各硬件状态
